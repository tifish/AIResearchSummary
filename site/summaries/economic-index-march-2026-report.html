学习曲线：为什么"老用户"用 Claude 更成功

这期报告的真正贡献，是第一次用真实使用数据证明——"会用 AI"本身是一种随经验积累、且分布极不均匀的技能。注册满半年的资深用户不仅敢接更高价值、更复杂的任务，对话成功率还实打实高出约 10 个百分点；即便控制掉任务类型、语言、国家等所有可见因素，仍剩下约 4 个百分点的净优势。这把去年"越老练越倾向全自动委托"的猜想推翻了：最资深的用户反而更爱和 Claude 来回协作。数据来自 2026 年 2 月 5–12 日各 100 万条 Claude.ai 与一方 API 对话。

Claude.ai 的任务变得更分散：体育比分、产品比价、家居维修这类简单事实型提问增多，把平均任务"含金量"略微拉低，人类输入所需的平均受教育年限也从 12.2 年降到 11.9 年。但这并不意味着 Claude 在学"新活"——几乎所有任务此前都出现过，"约 49% 的职业已有至少四分之一任务被 Claude 做过"这一累计估计几乎没变。编程则在持续从 Claude.ai 向 API 迁移（Claude Code 把编程拆成大量细碎 API 调用）。

这是典型的"采用曲线"：早期用户偏好编程等高价值场景，后来者带来的是范围广得多的日常杂务；扩散在变广，而非在解锁全新能力。

02用户会"按任务挑模型"——越值钱的活越用 Opus

对比维度（付费 Claude.ai）用 Opus 的占比
计算机与数学类任务55%

软件开发者任务34%

人们并非随手选模型，而是把最复杂、最值钱的任务交给最强的 Opus，把简单任务留给更快更省的 Sonnet。量化看：任务对应时薪每多 $10，使用 Opus 的对话占比在 Claude.ai 上升 1.5 个百分点；而在 API 端这一斜率约是两倍（2.8 个百分点）——做程序化工作流的开发者对成本和复杂度更敏感，模型切换更果断。

"为任务匹配算力"是一种已经在真实付费行为里显现的成熟用法；API 用户比网页用户更精于这种性价比权衡。

把用户按注册时长分组后，差异很清晰：每多用 Claude 一年，能读懂其提示所需的受教育年限就增加近 1 年；个人用途占比则从最新用户的 44% 降到老用户的 38%。平均资历最高的任务簇是 AI 研究、Git 操作、修改稿件、创业融资；最低的是写俳句、查体育比分、给聚会想菜单。换句话说，老用户带来的是更"工作化、更复杂"的需求。

这推翻了"越资深越爱全自动委托"的旧假设——最进阶的用户其实更愿意和 Claude 反复迭代协作。

04这不只是"老用户碰巧更厉害"——回归排除了简单解释

回归设定高资历的成功率优势
(1) 仅"是否高资历"单变量约 5 个百分点
(2) 加入具体 O*NET 任务/请求簇固定效应降到约 3 个百分点
(3) 再控制模型、语言、用途、国家回升到约 4 个百分点

关键在于设定 (2)：它是在同一个细分任务内部比较高/低资历用户，因此排除了"老用户只是挑了更容易成功的任务"这一解释；加上全套控制后优势依然稳健。最可能的解释之一就是边做边学——用得越久，越懂得如何从 AI 身上"榨"出自己想要的结果。当然，队列效应和幸存者偏差还无法完全排除（早期采用者本就更技术、留存下来的多是受益者），但简单版本的混淆已被排除。

"会提示、会驾驭 AI"看起来是一项可随经验增长的真实技能，而不只是早期用户的固有属性。

美国各州继续收敛：前 5 州的人均使用份额从 2025 年 8 月的 30% 降到 2 月的 24%，但收敛节奏已明显放缓。与此相反，国家之间却更不均：使用最密集的国家占了更大份额，低采用国家略微进一步掉队。同期 API 端任务反而更集中（前 10 大 O*NET 任务占比 28%→33%），而商业销售外联、自动化交易与市场操作这两类自动化工作流的份额至少翻了一倍。

国内鸿沟在被 AI 抹平，国际鸿沟却被它固化——全球层面的不平等正被锁定而非弥合。

最值得带走的 5 条洞见

  1. "会用 AI"是一种技能，且会随经验增长。满 6 个月的用户成功率高约 10%，控制任务/语言/国家后仍剩约 4 个百分点净优势——证据指向"边做边学"。
  2. 资深用户更爱协作而非全权委托。这推翻了去年"越老练越自动化"的假设：最进阶的用户反而更倾向和 Claude 反复迭代、用于真实工作（+7 个百分点）。
  3. 用户已学会按任务挑模型。任务时薪每多 $10，Opus 使用占比在 Claude.ai 升 1.5、在 API 升 2.8 个百分点——为复杂、高价值的活匹配最强算力。
  4. 使用在变广、变"便宜"。前 10 大任务占比 24%→19%，平均任务时薪 $49.3→$47.9，个人用途 35%→42%——经典的"晚期采用者拉低均值"扩散曲线。
  5. 这可能加深不平等。早期高技能采用者既最可能被 AI 冲击，又在这一波增强式浪潮中最受 AI 助益——一条技能偏向型技术变革的现实通道正在显现。
