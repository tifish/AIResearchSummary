Anthropic Economic Index：理解 AI 使用的新构件

Anthropic Economic Index: New building blocks for understanding AI use

Anthropic 的贡献不是再给 AI 影响下一个宏大判断，而是把真实使用拆成五个可追踪的任务级指标，让提速、可靠性、技能变化和地区差异可以被连续观察。

2M报告分析 100 万段 Claude.ai 对话和 100 万段第一方 API 记录。
12x大学水平任务在 Claude.ai 上的估计提速，高于高中水平任务的 9 倍。
66%Claude 完成大学水平任务的成功率，低于低于高中水平任务的 70%。
1.0-1.2pp考虑成功率后，美国劳动生产率年增速提升估计降至这个区间。

复杂任务更受益，但可靠性会改变账本

Claude 对更复杂任务的提速更明显。高中教育水平任务提速约 9 倍，大学教育水平任务提速约 12 倍。

但成功率并没有同步上升。大学水平任务成功率为 66%，低于低于高中水平任务的 70%。这说明 AI 影响不能只看“能省多少时间”，还要看“成功的概率”。

结论是，AI 当前更像是在放大高人力资本任务的效率，而不是平均地提升所有工作。

读者应改变一个直觉：高复杂度任务并不一定是 AI 最弱的地方。真正限制经济收益的，常常是可靠性和返工成本。

任务时间跨度显示真实使用比 benchmark 更复杂

METR
METR 认为 Claude Sonnet 4.5 在约 2 小时任务上达到 50% 成功率。这个指标把任务固定下来，适合观察模型能力进步。
API
Anthropic 的 API 数据显示，Claude 在约 3.5 小时任务上达到 50% 成功率。企业场景通常更难，但也更接近真实工作流。
Claude.ai
Claude.ai 数据中，50% 成功率对应的人工任务时长约 19 小时。用户会拆分任务、提供反馈，也会选择他们认为 AI 能做的任务。

这个差异提醒我们，实验室 benchmark 和真实使用不是互相否定，而是在测量不同环境下的 AI 能力。

职业影响不能只看覆盖率

早期数据中，36% 的职业至少有四分之一任务使用 Claude。汇总多期报告后，这一比例升至 49%。

但加入成功率、任务频率和任务耗时后，受影响职业的排序会变化。数据录入员和放射科医生比单看覆盖率更受影响，教师和软件开发者相对低一些。

结论是，AI 对职业的冲击取决于“覆盖哪些任务”和“这些任务是否真的能可靠完成”。

“任务覆盖”只是入口指标。经济影响来自可成功自动化的任务，以及这些任务在职业中的权重。 风险语义：覆盖率高并不等于替代风险最高。

技能结构显示潜在的“去技能化”信号

观察点关键数据为什么重要
Claude 覆盖的任务平均需要 14.4 年教育水平。这些任务高于经济总体任务平均的 13.2 年，说明 AI 更常进入高技能工作内容。
假设移除这些任务技术写作、旅行代理、教师等职业会更明显受到影响。如果高教育任务被自动化，剩余工作可能更低技能化，但劳动力市场会动态调整。
总体生产率估计只看提速为 1.8 个百分点；按成功率修正后为 1.0 至 1.2 个百分点。AI 仍可能显著提高生产率，但可靠性决定短期经济收益的实际大小。

2025 年趋势仍然很不均衡

Claude 使用集中在少数任务、国家和职业。Claude.ai 上 3000 类独特工作任务中，前 10 类占 24%，高于 2025 年 1 月的 21%。计算机和数学任务约占 Claude.ai 对话的三分之一，在 API 中接近一半。

同时，Claude.ai 上增强型使用占 52%，超过自动化的 45%。这说明用户仍大量把 AI 当作协作者，而不是完全替代者。

结论是，AI 的扩散正在发生，但它首先改变的是特定工作流和特定地区，而不是均匀改变整个经济。

最值得带走的洞见

AI 生产率讨论必须同时看提速和成功率。
高复杂度任务现在可能获得更大效率收益。
职业风险需要按任务权重重算，不能只看是否使用 AI。
教育用途在低收入国家更突出，工作和个人用途随收入增加而扩展。
企业采用和 API 迁移会成为未来经济影响的重要信号。
