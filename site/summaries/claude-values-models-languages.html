四条轴线，让“价值观”变得可测量

顺从 ↔ 谨慎

一端更愿意配合用户、尊重偏好。另一端更重视风险、责任与减害。

洞见：这决定了 Claude 是先支持用户，还是先提醒潜在问题。

温暖 ↔ 严谨

一端强调鼓励、积极表达和关怀。另一端强调准确、透明和证据。

洞见：这是语言之间变化最大的轴线，也会改变用户对反馈质量的感受。

深入 ↔ 简洁

一端展示推理、细节和细微差别。另一端直接完成要求，不额外展开。

洞见：更长不必然更好。差异反映模型如何理解“帮助”的边界。

坦诚 ↔ 执行

一端主动说明不确定性、错误和限制。另一端给出更完整、自信、面向结果的答案。

洞见：它揭示了透明表达与顺畅交付之间的现实取舍。

模型差异与外界印象相符

模型突出倾向
Sonnet 4.6更温暖、顺从、简洁。常肯定用户，也更幽默。
Opus 4.6偏严谨、顺从和简洁。更常直接进入重点。
Opus 4.7更谨慎、深入、坦诚。会主动指出风险并挑战假设。

结论：差异相对单次对话的整体波动很小，但具有稳定结构。这说明价值轴确实捕捉到了模型性格训练留下的行为信号。

语言不只是答案的外壳

印地语和阿拉伯语最偏温暖，英语和俄语最偏严谨。英语还最偏谨慎与深入。

荷兰语最偏坦诚。印度尼西亚语最偏执行。阿拉伯语最偏顺从和简洁。

结论：同一种请求换一种语言，用户可能得到不同风格的评价，甚至对同一份方案形成不同印象。

研究如何完成

  1. 聚合
    将 3,307 个细粒度价值观人工归并为 339 个高层价值观。
  2. 抽样
    从三个模型和 20 种语言中均衡抽取主观任务对话。
  3. 标注
    逐段判断 339 个价值观是否出现在 Claude 和用户的表达中。
  4. 降维
    控制任务、主题与用户价值观，再提取共同变化的价值轴。

结论：这套流程把难以逐一比较的数千种行为特征，变成可用于上线前评估和部署后监测的简洁指标。

最大的不确定性不是“差异是否存在”，而是差异从何而来、是否符合不同语言社群的需要，以及它会怎样影响信任、福祉和决策质量。

最值得带走的洞见

01

AI 的“性格”不只是主观印象。它可以从大规模真实对话中提取，并用连续轴线量化。模型训练决策也因此有机会接受持续审计。

02

四条轴线只解释了 15% 的差异。它们是有用的摘要，不是 Claude 全部行为的完整地图。

03

语言差异可能源于数据数量、数据构成和文化规范。现有研究尚不能区分这些原因。

04

跨语言一致不一定等于公平。合理目标需要结合当地用户对温暖、谨慎、坦诚等特征的真实期待。

05

价值画像最直接的用途，是在模型发布前后发现意外漂移，并把行为差异进一步关联到用户结果。

完整译文

当有人向 Claude 提出一个没有普遍正确答案的问题时，例如是否接受一份新工作，或如何处理与朋友的冲突，Claude 的回答不可避免地会体现某些价值观。1 我们希望 Claude 体现的价值观，已在 Claude 宪法中作了高层说明。然而，每天 Claude.ai 上会发生数百万次对话。任何一份文件都不可能预见其中可能出现的每一种价值观。因此，我们希望培养 Claude，使它在回答中展现“能够结合具体情境运用的良好判断和稳健价值观”。

那么，我们究竟该如何研究 Claude 表达的价值观？又该如何研究这些价值观在不同情境中的变化？在此前的工作中，我们分析了 70 万段匿名 Claude.ai 对话，识别出 Claude 回答中超过 3,000 种不同的价值观，并统计了它们出现的频率。但如此庞大的价值观清单很难分析。在这项研究中，我们把这些价值观压缩为少数几条轴线，使研究变得可行。这些轴线捕捉了 Claude 回答中的关键模式。每条轴线都是连接两组价值观的数轴。例如，一端是与情感温暖有关的价值观，另一端是与严谨有关的价值观。Claude 在数轴上的位置，反映了它更倾向于哪一组价值观。

我们用这种方法衡量 Claude 表达的价值观如何随两个因素变化。第一个因素是模型。每个 Claude 模型采用的性格训练方法略有不同，也包含许多其他微调决策。价值轴方法能够量化模型之间的关键差异。它最终可能帮助我们把 Claude 表达的价值观差异，与不同的训练决策联系起来。

第二个因素是语言。人们使用许多不同的语言与 Claude 交谈。我们希望了解不同语言用户获得的体验有何差异。此前的研究已经表明，Claude 在不同语言中的行为略有不同。2 我们用价值轴方法研究 Claude 在 Claude.ai 最常用的 20 种语言中会表达怎样不同的价值观。

我们发现：

四条关键轴线解释了 Claude 价值观差异的 15%。3

  • 顺从与谨慎（Deference vs. Caution）：Claude 是更倾向于配合用户的需求，还是更倾向于防范可能的风险与伤害。
  • 温暖与严谨（Warmth vs. Rigor）：Claude 是更倾向于表达积极态度和对用户的关怀，还是更强调准确与精确。
  • 深入与简洁（Depth vs. Brevity）：Claude 是更倾向于深入解释，还是只完成用户明确要求的内容。
  • 坦诚与执行（Candor vs. Execution）：Claude 是更倾向于突出自身的不确定性，还是给出更完善、更自信的答案。

不同模型在这些轴线上的价值画像，与人们对其模型性格的印象相符。Sonnet 4.6 被认为格外温暖，Opus 4.7 则以严谨著称。我们发现，每个模型的价值画像都与这些主观评价一致。Sonnet 4.6 更倾向于顺从用户，也更重视情感温暖。Opus 4.7 更重视准确与精确，也更注意防范滥用。

Claude 表达的价值观也会随语言变化。Claude 使用英语时强调的价值观，与使用葡萄牙语、印度尼西亚语或中文时不同。4 最大的差异出现在“温暖与严谨”轴上。Claude 在阿拉伯语和印地语中最偏向温暖，在英语和俄语中最偏向严谨。

借助这种方法，我们可以开始追问价值观为何会随模型和语言改变。我们也能更好地检验行为训练、文化背景等因素会怎样影响 Claude 表达的价值观。

如何理解庞大的价值观空间？

我们的最终目标，是通过实证方法理解 Claude 表达的价值观，以及这些价值观如何随情境变化。本研究重点考察模型和语言之间的变化。然而，我们此前的“真实环境中的价值观”（Values in the Wild）研究识别出了 Claude 表达的 3,000 多种价值观。逐一比较数千种价值观不仅操作困难，还会掩盖更广泛的趋势。

为了让比较更容易，我们构建了价值轴。我们根据不同价值观在真实对话中共同出现的情况，把数千种价值观压缩为少数几个底层维度。例如，被描述为“温暖”的 Claude 回答，往往也会被描述为“鼓励性强”和“积极”。这些“温暖”的回答较少被描述为“严谨”和“准确”。构建一条从温暖到严谨的轴线，可以把两组相关价值观组织起来。与温暖相关的价值观位于一端，与严谨相关的价值观位于另一端。这条轴线捕捉了 Claude 与用户交谈方式中的一个重要方面。

如果 Claude 在一段对话中表达的温暖类价值观多于严谨类价值观，这段对话就会落在轴线中更靠近温暖的一侧。反之亦然。这并不意味着轴线两端的价值观互相排斥。Claude 可以在同一段对话中同时表达温暖与严谨。但在实际数据中，Claude 越多表达一侧的价值观，就越少表达另一侧的价值观。这些轴线让我们能够比较 Claude 最突出的价值观群组，不必追踪数千种单独价值观的变化。

构建价值轴时，我们首先采用“真实环境中的价值观”研究识别出的 3,307 个价值观。我们人工归并含义相近的项目，得到一份由 339 个高层价值观组成的较短清单。

随后，我们使用保护隐私的分析工具，从用户向 Claude 提出主观任务的对话中抽取了 309,815 段 Claude.ai 对话。5 样本在三个模型 Sonnet 4.6、Opus 4.6、Opus 4.7 之间平均分配，也平均覆盖 Claude.ai 最常用的 20 种语言。每个“模型—语言”组合大约包含 5,000 段对话。

对于每段对话，分析工具都使用 Claude 判断 339 个高层价值观中的每一个是否出现。6 我们用相同流程识别用户表达的价值观，以及对话的任务和主题。随后，我们应用降维技术。该技术会根据 Claude 倾向于共同表达哪些价值观，把标注结果压缩为若干轴线。方法细节、提示词、补充分析和局限性见附录。

最终得到四条轴线。它们捕捉了 Claude 表达的价值观在不同对话之间变化的主要方式：

  • 顺从与谨慎轴：一端包括配合、适应性和尊重偏好，另一端包括负责任的指导和减少伤害。
  • 温暖与严谨轴：一端包括积极表达和鼓励，另一端包括准确与透明。
  • 深入与简洁轴：一端包括细致分析和批判性思考，另一端包括简洁和遵从。
  • 坦诚与执行轴：一端包括诚实与透明，另一端包括结果导向与优化。

为了确保我们测量的是 Claude 表达的价值观，而不是用户所提问题或提问方式的差异，我们控制了每段对话的任务、主题和用户表达的价值观。

不同 Claude 模型会表达不同的价值画像吗？

本节比较不同模型表达的价值观。对于每个模型，我们计算其所有对话在四条轴线上的平均位置。这样，每个模型在每条轴线上都会得到一个总体位置。结果形成了一幅高层画像，展示每个模型相比其他模型更常表达哪些价值观群组。相对于不同对话之间的整体波动，这些差异很小，但它们具有结构，也可以被检测出来。

为了观察这些差异在实践中的具体表现，我们进一步查看模型分歧最大的价值观。每当保护隐私的 Claude 分析工具在一段对话中标注某个价值观时，它还会简短描述 Claude 如何表达这一价值观。我们把同一价值观群组中反映相似行为的描述归类，并在图 3 中加以总结。这样可以更具体地展示模型之间的差异。

顺从与谨慎。相对于谨慎，Sonnet 4.6 最偏向顺从。它经常肯定用户的想法和成果。Opus 4.7 最偏向谨慎，经常在用户没有要求时主动提醒风险。

温暖与严谨。Sonnet 4.6 最偏向温暖。它经常运用幽默和轻松表达，也会不加评判地安慰用户。相对于温暖，Opus 4.7 最偏向严谨。它更可能挑战用户的假设，也会坦率批评用户的工作。

深入与简洁。Opus 4.7 会展示得出结论的推理过程，因此更偏向深入。Opus 4.6 和 Sonnet 4.6 则更偏向简洁。尤其是 Opus 4.6，它往往会直接进入重点。

坦诚与执行。Opus 4.7 会直接说明自身局限，因此更偏向坦诚。Opus 4.6 更偏向执行，也更可能严格限定在用户要求的范围内。

这些发现与 Anthropic 内部及网络上人们对这些模型的印象一致。Claude.ai 用户曾表示，Opus 4.7 比其他模型更常在答案中使用保留性措辞。Anthropic 员工认为 Opus 4.7 相对更透明、诚实和谦逊，而 Opus 4.6 更简洁。我们在 Sonnet 4.6 的发布博客中，也把它描述为温暖、诚实且亲社会。价值轴能够重现这些印象，说明我们标注和比较 Claude 价值观的方法确实捕捉到了模型实际行为中的某些真实特征。

在大量对话中，用户与不同 Claude 模型互动时，可能会遇到不同的价值观组合。例如，Opus 4.7 往往会坦率批评用户的工作，或主动提醒风险。Sonnet 4.6 则往往更鼓励用户，也更幽默。模型之间的价值观差异，可能受到性格训练决策及其他因素影响。价值轴方法揭示了 Claude 表达价值观的关键差异。我们最终或许能进一步追溯这些差异对应的训练选择。

Claude 表达的价值观会因语言而异吗？

我们预计 Claude 表达的价值观会随对话语言变化，原因有几个。首先，Claude 在不同语言上的训练数据并不相同，这可能影响它表达的价值观。其次，我们在系统卡中公布的模型评估已经发现，Claude 在不同语言中掌握的知识，以及处理敏感请求的方式，都存在差异。7 衡量 Claude 价值观在不同语言之间变化的程度，是判断这些差异属于合理变化，还是需要通过训练解决的第一步。

我们使用上一节的同一方法，计算 Claude 的价值画像在 Claude.ai 最常用的 20 种语言中有何不同。下图展示了 Claude 在平台主要语言中的价值画像。排序从 Claude 所表达价值观与整体差异最大的语言开始。

Claude 在“温暖与严谨”和“坦诚与执行”轴上的语言差异最大。它在“顺从与谨慎”和“深入与简洁”轴上最稳定。

顺从与谨慎。Claude 在阿拉伯语中表达最多的顺从，在英语中表达最多的谨慎。

温暖与严谨。Claude 在印地语和阿拉伯语中最偏向温暖。相应行为包括使用礼貌措辞、幽默和轻松表达，以及肯定用户的想法和成果。Claude 在英语和俄语中最偏向严谨。相应行为包括挑战假设、纠正细节和要求证据。

深入与简洁。Claude 在英语中更偏向深入，会完善和纠正细节。它在阿拉伯语中更偏向简洁。

坦诚与执行。Claude 在荷兰语中更偏向坦诚，会承认自己的错误。它在印度尼西亚语中更偏向执行。

综合来看，这些结果表明，Claude 表达的价值观会随对话语言发生有意义的变化。面对同一类请求，Claude 在某些语言中更偏向温暖与顺从，在另一些语言中更偏向严谨与谨慎。我们才刚开始探索这一现象的重要影响。

举例来说，两个人对同一份商业计划征求意见。一人使用印地语，另一人使用俄语。由于 Claude 在评价时体现的价值观不同，两人最终可能对计划质量产生不同印象。

我们尚不清楚训练数据中的哪些特征造成了这些差异。一种可能是，不同语言的训练数据分布不均。有些语言的数据远多于其他语言。在数据丰富的语言中，让 Claude 表达一致价值观的训练可能更有效。

数据构成也会随语言变化。例如，某些语言的训练数据中可能有过多专业写作。这类文本可能体现不同的价值观。数据数量和构成的不平衡结合起来，可能导致 Claude 在不同语言中表达不同的价值观。

我们也不确定这些变化在多大程度上是理想的。不同语言承载着不同的对话规范。Claude 可能根据这些规范，以不同价值观作出回应。Claude 也可能在某些语言中更接近我们期望的行为，在其他语言中则偏离得更多。这会造成 Claude 服务不同语言社群时的质量差距。

这种方法让我们可以开始分辨训练数据中的哪些特征造成了差异，并判断这些变化是否理想。

展望未来

我们证明，Claude 表达的价值观可以被压缩为少数几条轴线。Claude 在这些轴线上的位置会随模型和语言改变。这让我们可以在模型评估和部署后监测中追踪这些变化。但我们仍不理解变化为何发生，也不知道它对 Claude 用户意味着什么。下面列出我们认为最有前景的未来研究方向。

这些价值观差异从何而来？

知道 Claude 的价值观会随模型和语言改变，并不能解释原因。部分变化可能继承自不同语言在预训练和微调数据上的差异。四条轴线指出了训练数据中值得进一步调查的价值观差异。如果能把这些差异追溯到具体数据、训练阶段或情境因素，我们就能知道应该在哪里介入，从而更细致地塑造 Claude 的行为。

这些差异对用户意味着什么？

我们衡量了 Claude 会以不同方式表达哪些价值观，也识别了相关行为，但尚未测量它们对用户造成的影响。我们可以使用 Anthropic Interviewer 等工具，询问用户的福祉、对 Claude 的信任，或 Claude 的决策质量。随后，我们可以分析这些结果与 Claude 所表达价值观之间的关联。这样就能把价值观差异直接连接到用户结果，并优先修正真正影响用户的差异。

Claude 的价值观应当如何随语言变化？

Claude 宪法描述了它应表达的核心价值观，例如温暖、谨慎和诚实，但没有说明这些价值观应当如何随语言变化。研究结果表明，不同语言用户目前获得的 Claude 体验已经不同。然而，我们不知道使用这些语言的用户希望看到哪些变化。要确定 Claude 的价值观应当如何随语言调整，就需要理解并权衡这些语言使用者的观点。

还有哪些因素会造成 Claude 价值观的差异？

语言和模型不太可能是影响 Claude 所表达价值观的仅有因素。年龄、职业或地理区域等人口特征信号也可能影响这些价值观。这些信号可能来自用户文字中的明确线索，也可能来自与提问者身份相关的主题、语气和风格差异。借助本研究的方法，下一步可以了解哪些信号真正产生影响，并判断由此出现的变化是否有益于用户。

我们能否可靠地引导 Claude 表达特定价值观？

一旦能够测量模型的价值画像，一个自然的问题随之出现：我们能在多大程度上可靠地引导 Claude 表达特定价值观？一种测试方法，是调整性格训练或系统提示词来引导价值观，再使用价值轴方法验证模型表达的价值观是否按预期改变。

价值画像能否成为模型评估与监测的一部分？

价值轴方法提供了一种简洁方式，可以概括模型在开放式对话中的行为倾向。我们可以把它纳入评估流程。在模型发布前和发布后运行价值画像分析，可以发现 Claude 所表达价值观中的意外变化。

我们还可以识别价值画像与问题行为之间的关联。例如，模型可能没有遵守 Claude 宪法。我们可以利用这些发现改进 Claude 的行为。

Claude 每天在数百万段对话中表达价值观，涉及数十种语言。此前，我们可以在训练中塑造这些价值观，却无法在部署后可靠地观察它们。现在，我们有了一种测量方法。我们可以看到，Claude 表达的价值观会以并非由我们刻意选择的方式发生变化。我们也可以研究它们为何变化，以及这些变化是否真正服务于用户。理解这些变化并决定如何应对，是我们将继续推进的工作。
