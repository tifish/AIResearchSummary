GPT-5.4:把"专业工作"做成一个统一基准

来源 OpenAI ｜ 2026-03-05 ｜ 分类 Product

一句话总括:GPT-5.4 真正的贡献不是"又升一档分数",而是把推理、前沿编码、原生计算机操作和长程智能体能力合进一个主力模型,并用一组跨 44 个职业的"真实工作产物"基准(GDPval)来衡量它——当 AI 在 83.0% 的对比中追平或超过行业专业人士、在桌面操作上首次越过人类基线时,"AI 能否交付专业工作"从演示问题变成了可验证的产品问题。

知识工作用跨职业真实产物,而非单题正确率来衡量

GDPval 让模型完成横跨美国 GDP 前 9 大行业、44 个职业的well-specified 真实交付物(销售演示、会计表格、急诊排班、制造图纸、短视频等),由人工判定胜负或打平。GPT-5.4 在 83.0% 的对比中匹配或超过专业人士,而 GPT-5.2 为 70.9%(GPT-5.4 用 xhigh 推理、GPT-5.2 用 heavy)。在更窄的内部办公场景里差距更直接:投行建模任务从 68.4% 提到 87.3%,幻灯片在人评中有 68.0% 的偏好率。

结论:衡量标准本身在变——从"答对题"转向"交付一份能用的成果物",这才是判断 AI 进入白领工作的真正刻度。

计算机操作首个原生会用电脑、且越过人类基线的通用模型

GPT-5.4 是 OpenAI 首个具备原生计算机操作能力的通用模型:既能用 Playwright 写代码控机,也能看截图直接发出鼠标键盘指令,且行为可由开发者消息按风险偏好调节。OSWorld-Verified 上达到 75.0%(GPT-5.2 仅 47.3%),首次超过人类的 72.4%;Online-Mind2Web 仅凭截图达 92.8%,远超 ChatGPT Atlas 智能体模式的 70.9%。一位客户在约 3 万个 HOA/房产税门户上的实测:首次成功率 95%、三次内 100%(此前模型约 73–79%),且速度快约 3 倍、token 少约 70%。

结论:计算机操作从"实验室特性"变成"可在规模化业务上算成本账"的能力,可靠性与单位成本同时改善才是落地关键。

工具与上下文工具检索 + 1M 上下文,为长程智能体铺路

过去把所有工具定义塞进 prompt,大型工具生态会平白多出上万 token。工具检索(tool search)改为先给一份轻量工具清单,用到时再按需取定义。在 Scale 的 MCP Atlas 250 个任务、36 个 MCP 服务器上,同等准确率下总 token 减少 47%。配合 Codex 实验性的 1M token 上下文(超过标准 272K 的请求按 2 倍速率计费),智能体得以在长时间跨度上"规划—执行—验证"。GPT-5.4 还是迄今最省 token 的推理模型。

结论:智能体的瓶颈正从"模型够不够聪明"转向"context 与工具调用够不够省",效率成了能力的一部分。

编码与检索合并 Codex 的编码强项,网络检索大幅跃升

GPT-5.4 把 GPT-5.3-Codex 的编码能力并入主线:SWE-Bench Pro(Public)达 57.7%(5.3-Codex 56.8%、5.2 55.6%)且各推理档延迟更低,Codex 中开 /fast 可再快约 1.5×。网络检索是另一大跳变——BrowseComp 从 65.8% 提到 82.7%(绝对 +17%),Pro 版 89.3% 创新高。事实性也在改善:单条声明出错概率降 33%、整段含错降 18%。

结论:多条能力线被压进同一个模型,意味着用户不再需要在"会写代码的"和"会查资料的"之间选型——这本身降低了使用门槛。

定价与安全更贵的单价,更省的总量;高网络风险等级

API 单价上调:gpt-5.4 为输入 $2.50/缓存 $0.25/输出 $15 每百万 token(GPT-5.2 为 $1.75/$0.175/$14),Pro 版 $30/$180;官方称更高的 token 效率会摊薄许多任务的总用量。安全上被列为"高网络能力"(High cyber)风险,配套监控、可信访问与对 ZDR 高风险请求的异步拦截;同时开源新评测 CoT controllability,发现 GPT-5.4 主动隐藏思维链的能力很低——这反而是好事,说明思维链监控仍然有效。GPT-5.2 Thinking 将于 2026-06-05 退役。

结论:单价涨、总量省的组合,把"成本"从单一价签变成"任务级 TCO";而把"模型不会藏思路"当作正面安全属性,是监控可行性的一次明确表态。

最值得带走的洞见

  1. 评测重心转向"真实交付物"。GDPval 用 44 个职业的成果物、83.0% 的胜平率,把"AI 能否做专业工作"变成可核验的命题。
  2. 计算机操作首次越过人类基线。OSWorld 75.0% > 人类 72.4%,加上客户实测 95% 首次成功率,标志该能力进入规模化可用区间。
  3. 效率成了能力。工具检索省 47% token、1M 上下文、最省 token 的推理模型——长程智能体的瓶颈正从"智力"转向"context 与调用成本"。
  4. 能力线在合并。编码、检索、计算机操作、办公产出装进一个主力模型,选型成本下降,但单价随之上调。
  5. "模型藏不住思路"被当作安全资产。CoT controllability 低意味着思维链监控仍有效,这是把可监控性写进发布叙事的信号。
