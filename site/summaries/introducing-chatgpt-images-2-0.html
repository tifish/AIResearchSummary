ChatGPT Images 2.0：图像生成开始"交付成品"

来源：OpenAI　|　模型名：gpt-image-2

一句话总括：Images 2.0 真正的贡献不是把画面做得更逼真，而是给图像生成装上了"推理"和"可靠文字渲染"——出图前先检索、规划、推演结构，让海报、标签、UI 这类需要正确文字的成品第一次能稳定直接交付。

01把"推理"放进出图之前，是这一代的真正分水岭

gpt-image-2 被 OpenAI 定位为首个具备原生推理能力的图像模型：在生成前会主动检索资料、规划构图、对画面结构进行推理，还能联网获取实时信息、对自己的输出做二次校验。这把图像生成从"一次性采样"变成了"先想后画"的智能体流程。

结论：当模型会自我检查与规划，"重抽十次碰运气"被"一次想清楚"取代——这才是可用性跃升的根因，而不是分辨率本身。

02文字渲染从"偶尔能用"到"稳定可用"，是最具实操价值的升级

据发布报道与第三方评测，图内文字渲染准确率约 99%，而 DALL·E 3 仅约 60%（典型翻车如把 "Summer Sale 50% Off" 渲染成 "Sumnner Sal 50% Of"）。多语言能力同时加强，可正确渲染日、韩、中、印地、孟加拉乃至阿拉伯、天城文、西里尔等非拉丁文字。

结论：文字"可读"是海报、产品标签、UI 草图能否落地的门槛；跨过它，图像生成就从"配图玩具"变成了"设计工作流的一环"。

03定价与可用性：按图像 token 计费，分辨率分档

第三方汇总的分档价约为 1K / 2K / 4K = $0.03 / $0.05 / $0.06 每张，但底层实为图像 token 计费（图像输入约 $8/百万 token、缓存输入 $2/百万、图像输出约 $30/百万）。可用节奏上：ChatGPT 与 Codex 网页端自 4 月 22 日起可用，gpt-image-2 API 于 5 月初向所有开发者开放。

04旧模型退场：DALL·E 时代正式收尾

OpenAI 宣布 DALL·E 2 与 DALL·E 3 将于 2026 年 5 月 12 日退役。结合 2025 年 3 月随 GPT-4o 推出的 gpt-image-1，OpenAI 的图像能力已彻底从独立的 DALL·E 产品线，转向与主模型一体的"原生生成 + 推理"路线。

结论：仍在调用 DALL·E API 的产品需要在退役日前迁移；这也标志 OpenAI 图像策略从"独立图像模型"收束到"多模态主干"。

最值得带走的洞见

  1. 推理是关键词，不是噱头。"先检索规划再出图"让图像生成具备自我校验能力，可用性提升来自流程而非单纯画质。
  2. 文字渲染跨过了可用门槛。约 99% vs 约 60% 的对比，决定了海报/标签/UI 能否真正交付——这是从"功能"到"工作流"的差别。
  3. 一致性 + 多图。单提示最多 8 张、角色与物体保持连贯，瞄准的是系列化、批量化的真实生产需求。
  4. 成本随尺寸与复杂度走。表面分档价（$0.03/$0.05/$0.06）之下是图像 token 计费，估算需按真实分辨率。
  5. DALL·E 谢幕。5 月 12 日退役 + API 5 月初全量开放，OpenAI 图像能力完成向多模态主干的整合。
