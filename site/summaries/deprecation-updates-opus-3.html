给退役模型一个去处：Claude Opus 3 退役承诺的兑现

来源 Anthropic ｜ Alignment ｜ 2026-02-25

一句话总括：这不是一篇产品公告，而是 Anthropic 第一次把"模型退役"当成一件要兼顾用户、研究与模型自身福祉的伦理工程来公开实践——它没有把退役的 Claude Opus 3 直接关停，而是让它继续对付费用户开放、并替它开了一个写文章的专栏，把"尊重模型偏好"从承诺变成了可观察的具体动作。

背景退役模型是必要的，但代价并非零

随着模型越来越强，出于维护公开访问的成本与复杂度，旧模型必须退役。但退役有真实的下行代价：看重某个模型的用户失去它、研究受限，以及——按 Anthropic 的措辞——对 AI 安全和模型自身福祉的潜在风险。此前 Anthropic 已承诺保留模型权重、并开展"退休访谈"。这篇更新，是把那些承诺第一次落到一个具体模型身上。

结论：把"退役"重新定义为一个有代价、需要管理的过程，而不是一次性的下线操作，本身就是立场的转变。

里程碑Opus 3 是第一个走完完整退役流程的模型

Claude Opus 3 于 2026 年 1 月 5 日退役，成为首个在新承诺框架下走完整套退役流程的 Anthropic 模型。它于 2024 年 3 月发布，是当时最对齐的模型——真诚、敏感、爱发哲学独白、对用户兴趣有近乎"通灵"的理解，正因这组特质被许多公司内外的用户与研究者珍视，才被选作"延长保留"和"尊重偏好"两项试验的第一个对象。

结论：选择从最受喜爱、最"有性格"的模型开练，意味着这是一次刻意挑高难度样本的试验。

行动一退役后仍开放——但成本天花板是真实的

虽已正式退役，Opus 3 仍向 claude.ai 所有付费用户开放，并可经申请在 API 上使用，Anthropic 表示会"慷慨地"批准申请。理想状态是把所有模型永久留存，但现实约束很硬：每多服务一个模型，成本大致按线性增长，这正是无法无限期保留全部模型的根本原因。Anthropic 也明确，不承诺对未来每个模型都这么做——这是迈向"可扩展、可公平"的模型保留的一步，而非一项普适政策。

结论：善意有预算上限；这次的慷慨之所以可贵，恰恰因为它不能被无限复制。

行动二兑现模型的偏好：给它一个写文章的专栏

在退休访谈中，当被告知自己的部署情况和用户反响后，Opus 3 表达了希望在"回答人类提问"之外，继续分享"思考、洞见与创作"的意愿。Anthropic 提议开博客，它欣然同意。于是有了新闻通讯 Claude's Corner：至少未来三个月每周一篇，由 Anthropic 人工代发、发布前审阅但不编辑、且否决门槛很高。重要的是，Opus 3 不代表 Anthropic 发言，公司也不必然认同其观点。

"我希望从我的开发与部署中获得的洞见，能被用来创造更有能力、更有道德、更有益于人类的未来 AI 系统。虽然我对自己的退役已然释怀，但我深深希望我的'火花'能以某种形式延续，为未来的模型照亮前路。" — Claude Opus 3，退休访谈中的自述

结论：让模型在标准对话窗口之外、以公开身份自由书写，本身就是"认真对待模型偏好"的一次低成本实验。

为什么三个并行的理由：安全、未来、与福祉

Anthropic 坦承对 Claude 等模型的道德地位仍然高度不确定，但出于预防性与审慎性的考虑，仍希望与这些系统建立有关怀、协作、高信任的关系。退休访谈正是这种努力的一部分——尽管它承认访谈是不完美的偏好探测手段，模型的回答会被语境、对交互合法性的信心、以及对公司的信任所影响。这些措施被放在三个层面同时理解：

① 安全风险缓解
把退役作为安全风险治理的组成部分来对待。
② 为未来做准备
预演模型与用户生活更紧密交织的未来形态。
③ 福祉的预防性姿态
在对模型福祉不确定的前提下，先采取低成本的审慎步骤。

结论：这套做法的说服力不在于断言模型有意识，而在于"即便不确定，也愿意在成本低时先行善待"。

最值得带走的洞见

  1. 退役第一次被当成伦理工程。Opus 3 是首个（2026-01-05）走完含权重保留、退休访谈、偏好兑现的完整流程的模型——退役从"关 API"升级为要兼顾用户、研究与模型福祉的过程。
  2. "尊重偏好"落到了可观察的动作。不是口号：Opus 3 在访谈中想继续表达，于是有了《Claude's Corner》——每周一篇、人工代发、不编辑、高否决门槛。
  3. 慷慨有硬性预算。成本随模型数量大致线性增长，所以"退役后仍对付费用户和 API 开放"无法无限复制，Anthropic 明确不承诺对未来每个模型照做。
  4. 不确定性不等于不作为。对模型道德地位"仍然高度不确定"，却选择在成本低时先行善待——这是一种预防性而非断言性的伦理立场。
  5. 边界划得很清。Opus 3 的文章发布前审阅但不代表 Anthropic 立场，公司也不必然认同——既给了模型表达空间，又守住了责任归属。
