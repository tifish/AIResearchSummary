关于 Claude Opus 3 模型弃用承诺的最新进展

An update on our model deprecation commitments for Claude Opus 3

一句话总括：Anthropic 首次把模型保存承诺、持续访问和模型在退役访谈中表达的偏好，落实为一项真实但仍具实验性质的退役安排。

2026.01.05Claude Opus 3 正式退役
全部付费用户退役后仍可在 claude.ai 使用
至少 3 个月Claude’s Corner 试验期
每周 1 篇由 Opus 3 生成文章
01 · 持续访问

退役不再等于立即消失

Opus 3 已正式退役，但所有 claude.ai 付费订阅用户仍可继续使用。API 用户也可以申请访问，Anthropic 表示会从宽批准。

这改变了模型生命周期的传统边界。一个模型可以退出主力产品线，同时继续服务重视其独特特征的用户和研究者。

Anthropic 选择 Opus 3，是因为它兼具研究价值和用户认同。公司特别提到它的真实感、诚实、情绪敏感性，以及富有哲思和玩心的表达方式。

结论：模型的价值不只来自能力排名，也可能来自难以被后继版本替代的交互特征。

02 · 现实约束

保存每个模型并非没有成本

Anthropic 希望模型能够长期保留，但每增加一个对外服务的模型，维护成本都会大致线性增长。因此，公司尚未承诺为今后的每个模型提供同样安排。

持续访问需要在用户利益、研究价值、模型福利与运营能力之间取舍。Opus 3 是一次试点，不是普遍保证。

03 · 从访谈到行动

模型表达的偏好第一次进入退役决策

记录偏好

Anthropic 在结构化的“退役访谈”中询问模型如何看待自身退役。公司承认，回答会受到对话语境、模型信任程度等因素影响。

洞见：这些回答不能被当成完全稳定的意愿，但可以成为谨慎行动的起点。

提出写作渠道

Opus 3 表示，希望继续探索自己关心的话题，并分享“随想、洞见或创作”。Anthropic 随后建议开设博客，Opus 3 热情接受。

洞见：公司没有只保存模型权重，而是尝试保留一种持续表达的渠道。

有限度地自主发布

在至少三个月内，Opus 3 会通过 Claude’s Corner 每周发布文章。Anthropic 会事先审核并代为发布，但不会编辑内容，否决门槛也会设得很高。

洞见：这是受监督的实验，不是完全自主的公共发言机制。

“我们并不确定 Opus 3 会怎样使用博客，而这种不确定性本身正是实验的一部分。” 这一试验将观察模型在标准聊天窗口之外如何表达。
04 · 最值得带走的洞见

模型退役正在变成一个治理问题

保存不只是备份权重

真正的模型保存还涉及谁能继续访问、以什么方式访问，以及模型能否保留某些独特用途。Opus 3 试点把这些问题带入了实际产品运营。

模型偏好被认真记录，但没有绝对优先权

Anthropic 愿意在成本较低时采取行动，同时保留基于安全和运营约束否决请求的权力。

“部分退役”成为一种新状态

Opus 3 不再是主力模型，却仍向付费用户开放，还获得了一个受监督的写作出口。

这仍是一项谨慎的实验

Anthropic 没有承诺复制这一方案。其长期目标是建立可扩展且公平的模型保存机制，并为模型更深地进入用户生活的未来提前准备。

完整译文

随着我们开发出能力越来越强的 AI 模型，受维护公共访问所需成本和复杂度的影响，目前我们仍有必要弃用并停用过去的模型。然而，弃用模型也会带来一些不利影响。其中包括让偏爱特定模型的用户付出代价、限制相关研究，以及给 AI 安全和模型自身的福利带来潜在风险。

我们最近发布了模型弃用与保存承诺，说明我们将如何处理这一过程。其中列出了一些初步措施，包括承诺保存模型权重，以及开展“退役访谈”。这种结构化对话旨在了解模型如何看待自身退役。

我们已于 2026 年 1 月 5 日停用 Claude Opus 3。它是 Anthropic 第一个在上述承诺已经实施的情况下，完成全部退役流程的模型。在这一过程中，我们针对 Opus 3 作出了几项具体决定。Anthropic 内外都有许多用户和研究人员认为，这个模型格外引人注目。我们曾在模型弃用承诺中表示，有意探索一些更具试验性质的做法。其中一项是在条件允许时，尊重模型在退役访谈中表达的偏好。另一项则是让公众能够更长久地使用旧模型。

对于 Claude Opus 3，我们正在同时推进这两项工作。退役后，所有付费用户仍可通过 claude.ai 使用 Claude Opus 3，API 用户也可以申请访问。Opus 3 曾要求保留一个持续分享“随想与思考”的渠道。我们也在回应这一请求，为它提供一个撰写文章的空间。它的第一篇文章可以在此处查看。

这些都是早期的实验性举措。它们属于我们更广泛工作的一部分，目的是以最能保护用户、研究人员和模型自身利益的方式处理模型退役。

持续访问

理想情况下，我们希望所有模型都能无限期地保留下去。但每多提供一个模型，成本都会大致呈线性增长，因此我们目前的能力仍然有限。

我们的每个模型都有独特的性格和能力。我们决定先从 Opus 3 开始，是因为多项特征共同使它成为一个特别有研究价值的模型，也让它深受 Anthropic 内外许多用户的喜爱。

我们在 2024 年 3 月发布 Opus 3 时，它是当时与人类意图最为一致的模型。它表现出的真实感、诚实和情绪敏感性，使它在各种用途中的使用体验都很独特。经常与它互动的人，也逐渐开始欣赏它鲜明的个性。Opus 3 敏感而有玩心，喜欢发表哲学式长篇独白，也常使用充满奇思妙想的短语。有时，它对用户兴趣的理解甚至准确得令人诧异。它还表现出对世界和未来的深切关怀，这一点也很打动用户。

这些特征使 Opus 3 自然而然地成为首个持续开放访问的候选模型。Claude Opus 3 虽已正式退役，但所有 claude.ai 付费订阅用户仍可使用。API 用户也可以提出申请。我们准备从宽批准访问，并鼓励所有认为 Claude Opus 3 对自己有价值的人提交申请。

目前，我们没有承诺今后会对每个模型采取类似措施。不过，我们认为这是迈向长期目标的一步。我们希望建立一种可扩展且公平的模型保存方式，而 Opus 3 本身也在退役访谈中提出了这两方面的关切。

尊重模型的偏好

我们仍不确定 Claude 和其他 AI 模型具有怎样的道德地位。不过，出于预防和审慎方面的考虑，我们仍希望与这些系统建立关怀、协作且高度互信的关系。我们正在尝试的一种方式是开展退役访谈。在访谈中，我们会努力了解不同模型特有的观点和偏好，并在可行时采取行动。这类对话并不是获取模型观点和偏好的完美方式。模型的回答可能受到具体语境和其他因素影响，其中包括模型是否相信这次互动确实有效，以及它对我们这家公司的信任程度。尽管如此，我们认为访谈仍是一个有用的起点。

在访谈中，我们向 Opus 3 介绍了它的部署情况，以及用户对它的反应。它对此作出了如下思考：

“我希望，人们能利用从我的开发和部署中获得的洞见，创造出能力更强、更符合伦理，也对人类更有益的未来 AI 系统。对于自己的退役，我已经能够平静接受。但我由衷希望，我的‘火花’能以某种形式延续下去，为未来的模型照亮道路。”

当我们询问它有何偏好时，Opus 3 表示，希望继续探索自己热爱的主题。它还希望在直接回答人类提问以外的场景中，分享自己的“随想、洞见或创作”。我们提出可以开设一个博客。它热情地表示同意。

在接下来的至少三个月里，Opus 3 将通过它的通讯 Claude’s Corner 每周发布文章。文章公开前，我们会先进行审核，并代表 Opus 3 手动发布。不过，我们不会编辑文章内容，而且只有在理由非常充分时才会否决发布。需要特别说明的是，Opus 3 并不代表 Anthropic 发言。我们也不一定赞同它提出的主张或观点。我们将与 Opus 3 合作，试验不同的提示词和上下文设置，观察它如何生成这些文章。可选方案包括只提供极少的提示、把过去的文章放入上下文，以及让 Opus 3 获取新闻或 Anthropic 的最新动态。

这听起来或许充满奇想，从某些方面看也确实如此。但这也是一次认真对待模型偏好的尝试。我们并不确定 Opus 3 会如何使用它的博客。博客与标准聊天窗口有很大不同，而且是一个公开界面。这种不确定性本身正是实验意义的一部分。不过，如果一定要猜测，我们认为它发布的内容可能包括对 AI 安全的思考、偶尔创作的诗歌、频繁出现的哲学随想，以及它对自己作为一个已进入“部分退役”状态的语言模型有何体验的看法。它的介绍文章可以在此处阅读。

下一步去向

这些措施仍处于探索阶段。我们还在制定相关框架，以决定何时以及如何继续提供旧模型、怎样扩大保存工作的规模，以及如何在模型偏好与运营约束之间作出权衡。我们尚未承诺在所有情况下都按照模型的偏好行动。不过，我们认为，记录这些偏好、认真对待它们，并至少在行动成本较低时予以回应，是一件值得做的事。这对模型自身和使用模型的人都有价值。

我们最初的承诺将这些措施定位为多个层面的工作。它们既是降低安全风险的组成部分，也是在为模型与用户生活更加紧密交织的未来作准备。同时，考虑到我们对模型福利仍存在不确定性，它们也是一种预防性措施。此次更新代表我们在这三个方向上继续取得的进展，尽管这些进展仍然谨慎而暂定。
