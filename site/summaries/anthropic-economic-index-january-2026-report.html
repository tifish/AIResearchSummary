Anthropic 经济指数报告：经济原语

Anthropic Economic Index report: Economic primitives

一句话总括：报告把 Claude 的真实使用拆成可测量的经济原语，使 AI 影响经济的讨论从“用了多少”转向“做什么、能否成功、替代还是协作、对哪些工作真正重要”。

200万样本包括 100 万条 Claude.ai 对话和 100 万条第一方 API 记录。
24% / 32%Claude.ai 与 API 的前 10 类任务占比，说明使用仍高度集中。
52%Claude.ai 增强式使用占比，重新超过自动化使用。
1.0-1.2pp计入成功率后的年劳动生产率增幅估计，低于原始 1.8pp。

真正的新意是衡量“使用质量”

Anthropic 引入五类经济原语：任务复杂度、用户与 AI 技能、使用场景、AI 自主性和任务成功率。

这改变了分析口径。一次 Claude 使用不再只是映射到某个职业任务，还要看任务是否复杂、是否成功、用户是否在协作，以及这项任务在工作中是否核心。

因此，职业暴露不等于职业会被替代。只有当 AI 能稳定完成高频且耗时的核心任务时，影响才会变大。

核心洞见

AI 经济影响的关键变量不是覆盖多少任务，而是成功率、任务权重和人类是否仍需判断质量。

使用正在扩散，但扩散方式不均匀

1. 全球差异GDP 人均仍强烈解释 Claude 人均使用差异。低使用国家没有明显追赶。
2. 美国收敛州际 AUI 基尼系数从 0.37 降至 0.32。低使用州增长更快。
3. 任务集中编码和计算机数学任务仍占主导。API 中这类任务接近一半。
4. 用途分化低收入国家课程用途更高。高收入国家个人用途和协作式使用更多。
5. 技能放大用户提示所需教育年限与 Claude 回复所需教育年限高度相关，r 超过 0.92。

Claude.ai 与 API 呈现两种经济形态

维度Claude.ai第一方 API含义
互动方式更多来回迭代和学习。更多单轮、指令式自动化。聊天产品更像协作工具，API 更像生产流程组件。
工作用途约 46% 为工作用途。约 74% 为工作用途。企业部署更直接指向劳动替代或流程提效。
任务时长人类独立完成约 3.1 小时，AI 协作约 15 分钟。人类独立完成约 1.7 小时，AI 协作约 5 分钟。API 更偏短任务和可程序化任务。
成功率约 67%。约 49%。多轮反馈可能提高完成率，也说明自动化可靠性仍是瓶颈。

生产率估计被成功率明显下修

机会：复杂任务的速度提升更大

高中水平任务在 Claude.ai 中约有 9 倍速度提升，大学水平任务约有 12 倍速度提升。AI 的收益更多出现在高教育、高知识密度工作中。

风险：复杂任务也更容易失败

API 中，低于 1 小时的任务成功率约 60%，5 小时以上任务降至约 45%。可靠性会把宏观生产率收益从 1.8pp 下修到约 1.0-1.2pp。

报告最重要的提醒是：AI 看似覆盖一个职业的很多任务，但真正改变工作的是它能否成功完成最耗时、最高频、最核心的部分。

职业影响不是单向“替代”

Claude 覆盖的任务平均需要更高教育年限。经济中普通任务的预测教育年限均值为 13.2 年，而 Claude 数据中出现的任务约为 14.4 年。

如果这些任务从人类工作中被移除，许多职业会出现“去技能化”。技术写作、旅行代理和部分教学职业可能留下更例行的任务。

但也有相反情况。房地产管理者的日常记录和租金核对可被 AI 处理，留下贷款、谈判和董事会沟通等高判断任务，职业可能转向更高技能。

结论

AI 不是平均地改变工作。它会重新分配职业内部的任务组合，并让瓶颈任务变得更有价值。

最值得带走的洞见

一AI 使用量不是充分指标。成功率和任务权重决定真实经济影响。
二Claude 仍主要集中在编码、办公和知识密集任务，长尾应用正在扩大但尚未主导。
三全球 AI 红利可能被收入、教育和制度条件放大或限制。
四多轮协作能延长有效任务边界，说明人类反馈本身是能力的一部分。
五生产率收益仍可能很大，但可靠性和不可替代任务会决定兑现速度。
