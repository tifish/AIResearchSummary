一句话总括

GPT-5.5 的真正贡献，是把模型能力推进到更接近“持续完成工作”的代理阶段。

文章的重点不只是 GPT-5.5 在若干榜单上更高分，而是它能在编码、研究、办公、电脑操作和安全任务中更稳定地理解目标、使用工具、迭代检查，并把复杂任务推进到可交付结果。

最重要的变化：模型更像能持续推进任务的同事

OpenAI 将 GPT-5.5 描述为更擅长处理“混乱、多部分、需要持续行动”的任务：它能规划步骤、调用工具、检查输出、在不确定性中继续推进，而不是等待用户逐步指挥。

这改变了读者对 AI 影响的理解：价值重心正在从一次性回答迁移到真实工作流中的执行能力，尤其是那些需要跨文件、跨工具、跨上下文反复判断的任务。

结论：GPT-5.5 的核心卖点是长链条自治能力，而不是单纯的聊天质量提升。

编码能力的提升最清晰，也最接近生产场景

在 Terminal-Bench 2.0 上，GPT-5.5 达到 82.7%，高于 GPT-5.4 的 75.1%；在 SWE-Bench Pro 上为 58.6%；内部 Expert-SWE 为 73.1%，也高于 GPT-5.4 的 68.5%。

这些评测关注的不只是写代码片段，而是命令行工作流、真实 GitHub issue、长期工程任务和工具协同。文章中的早测反馈也集中在系统理解、故障定位、分支合并、重构和测试判断上。

结论：如果这些结果能在真实使用中稳定复现，AI 编程的边界会从“辅助实现”扩展到“承接较完整的工程任务”。

知识工作不再只是生成文档，而是处理完整办公链路

GPT-5.5 在 GDPval 上达到 84.9%，OSWorld-Verified 为 78.7%，Tau2-bench Telecom 达到 98.0%。OpenAI 称它在生成文档、表格、演示文稿、信息检索、工具操作和结果校验上优于 GPT-5.4。

文章给出的内部案例很具体：OpenAI 每周使用 Codex 的员工超过 85%；财务团队用它处理 24,771 份 K-1 税表、共 71,637 页，较上一年节省约两周；市场团队有人每周节省 5-10 小时。

结论：企业采用 AI 的关键可能不再是“能生成什么文本”，而是能否嵌入现有流程并减少人工周转时间。

科研能力的亮点在于多阶段分析，而非只答难题

GPT-5.5 在 GeneBench 上从 GPT-5.4 的 19.0% 提升到 25.0%，GPT-5.5 Pro 达到 33.2%；FrontierMath Tier 4 从 27.1% 提升到 35.4%，Pro 为 39.6%；BixBench 达到 80.5%。

文章强调这些任务更像真实科研：处理有噪声的数据、发现混杂因素、做质量控制、实现统计方法并解释结果。案例包括分析 62 个样本、近 28,000 个基因的数据集，以及辅助发现并用 Lean 验证 Ramsey 数相关证明。

结论：科研场景中的 AI 价值正在从“解释论文”走向“参与实验设计、分析实现和论证检查”。

效率叙事同样关键：更强模型要能被服务出来

OpenAI 称 GPT-5.5 在真实服务中的每 token 延迟接近 GPT-5.4，同时在 Codex 任务中使用更少 token。它还与 NVIDIA GB200 和 GB300 NVL72 系统一起进行训练和服务协同设计。

一个具体基础设施改进是负载均衡与分区启发式算法：Codex 分析数周生产流量并生成定制算法，使 token 生成速度提升超过 20%。

结论：前沿模型竞争不只是模型本身，也包括推理系统、成本、延迟和服务工程的综合能力。

安全策略变严，尤其针对网络安全与生物化学能力

GPT-5.5 的网络安全能力被描述为高于 GPT-5.4，但未达到 Critical 等级；OpenAI 将其生物/化学和网络安全能力按 Preparedness Framework 视为 High。CyberGym 得分为 81.8%，高于 GPT-5.4 的 79.0%。

文章的政策重点是“可信访问”：对高风险网络活动、更敏感请求和重复滥用加强控制，同时给经过验证的防御者提供更少限制的 Trusted Access for Cyber。

结论：越强的代理模型越需要分层访问、身份信任和滥用监测，否则能力扩散会同时放大防御和攻击。

可用性与价格说明了它被定位为高端工作模型

GPT-5.5 面向 Plus、Pro、Business、Enterprise 用户在 ChatGPT 和 Codex 推出；Codex 中提供 400K 上下文窗口，并有 Fast mode：生成速度 1.5x，成本 2.5x。

API 标准价为 gpt-5.5 每 100 万输入 token 5 美元、每 100 万输出 token 30 美元；gpt-5.5-pro 为 30 美元和 180 美元。Batch 与 Flex 为半价，Priority 为标准价 2.5x。

结论：OpenAI 把 GPT-5.5 包装成更贵但更省步骤、更省 token、更适合高价值任务的模型。

最值得带走的洞见

  • 代理能力正在成为主战场。 GPT-5.5 的叙事核心是持续执行、工具使用和自我检查，而不是单轮生成。
  • 编码是最先被重塑的知识工作。 多项工程评测和早测案例都指向更强的系统理解、重构和端到端交付能力。
  • 科研和办公场景的共同点是多阶段判断。 模型价值来自把数据、工具、假设和输出串起来，而不是只给出答案。
  • 基础设施效率会决定能力能否规模化。 更高智能如果没有延迟、成本和吞吐改进，就难以进入日常工作流。
  • 安全访问会变成产品能力的一部分。 网络安全和生物化学能力越强，越需要可信用户、分级权限和持续监测。
