GPT‑5.5：把竞争从"考多少分"拉回"能不能放手交付"

来源 OpenAI  ·  日期 2026‑04‑23（API 于 4‑24 上线） ·  分类 Product

一句话总括：GPT‑5.5 真正的贡献不是某一项基准刷新，而是同时做到 更聪明、更省 token、且不掉速——在真实服务中保持与上一代 GPT‑5.4 相同的每 token 延迟，却能独立规划、调用工具、自检并把一个"杂乱的多步骤任务"一路做完。它标志着前沿模型叙事从"答对难题"转向"在长程真实工作中可被托付"。

01它最强的地方是"自主性"，而不是单点能力

智能体编码 · 计算机操作 · 长程任务

GPT‑5.5 在需要"跨上下文推理 + 持续行动"的场景提升最明显。Terminal‑Bench 2.0（复杂命令行工作流）拿到 82.7%，比 GPT‑5.4 的 75.1% 高出 7.6 个百分点，也明显领先 Claude Opus 4.7（69.4%）与 Gemini 3.1 Pro（68.5%）。在内部 Expert‑SWE 上——这些任务中位人工完成时间约 20 小时——它达到 73.1%。早期工程师反馈的关键词不是"更会写代码"，而是"理解系统的形状"：知道为什么坏、该改哪里、会牵连到哪里。

基准之外，多位资深工程师称其在"自主性与提前发现问题"上强于 GPT‑5.4 和 Claude Opus 4.7；一位 NVIDIA 早期用户甚至说"失去它像被截肢"。当模型能预判测试和评审需求、少需纠错时，人机协作的成本结构就变了。

02"更省 token"是这次最被低估的卖点

推理效率 · 成本

大模型通常更慢更贵；GPT‑5.5 的反直觉之处是：在三项编码评测全面超过 GPT‑5.4 的同时，用更少 token 完成相同任务。OpenAI 称在 Artificial Analysis 编码指数上，它以"竞品一半的成本"达到前沿智能。这背后是把推理当作整体系统来重构——模型与 NVIDIA GB200/GB300 NVL72 协同设计，甚至用 Codex 分析数周生产流量、自动改写负载均衡启发式算法，把 token 生成速度提升了 20% 以上。换句话说：模型帮忙改进了"服务自己"的基础设施。

"更强"如果同时意味着"更便宜、不更慢"，采用门槛会被显著拉低。效率本身正在成为前沿能力的一部分，而非事后优化。

03同样的能力外溢到了日常知识工作

文档 / 表格 / 计算机操作

因为更懂意图，GPT‑5.5 能走完"找信息→判断重点→用工具→检查→产出"的完整闭环。GDPval（覆盖 44 种职业的知识工作）得 84.9%，OSWorld‑Verified（自主操作真实电脑）78.7%，Tau2‑bench Telecom 在不做提示调优下达 98.0%。这些不再是抽象分数——它们对应"模型真的能和你一起用电脑"。

OpenAI 内部的真实工作流

· 财务团队用 Codex 审阅 24,771 份 K‑1 税表、共 71,637 页，比去年提速两周；

· 传播团队分析六个月演讲邀约数据，建评分/风险框架，让低风险请求由 Slack 智能体自动处理；

· 全公司逾 85% 员工每周使用 Codex，覆盖工程、财务、市场、数据科学等职能。

当"会编码"的底层能力被泛化成"会操作电脑、会处理杂乱业务输入"，AI 的影响范围就从工程师扩展到了几乎所有桌面工作者。

04科学研究：从"答题机"到"协作型科研伙伴"

基因 / 生物信息 / 数学

科研需要的是"探索→取证→验证→决定下一步"的持续循环，而非一次性答题。GPT‑5.5 在新评测 GeneBench（多阶段遗传与定量生物数据分析）从 19.0% 升到 25.0%（Pro 版 33.2%），BixBench（真实生物信息分析）达 80.5%——这些任务对专家而言常是"多天项目"。更具标志性的是：一个带定制 harness 的内部版本帮助发现了关于 Ramsey 数 的新证明（组合数学核心难题），并已在 Lean 中验证。

专家手中的具体加速

· 免疫学教授 Derya Unutmaz 用 GPT‑5.5 Pro 分析 62 个样本、近 28,000 个基因 的表达数据集，生成的研究报告"本来要团队数月"；

· 数学家 Bartosz Naskręcki 用 Codex 仅 11 分钟 从一个提示构建出代数几何应用，可视化二次曲面交线并转成 Weierstrass 模型。

当模型能把"专家意图"直接变成"可运行的研究工具与分析"，它在前沿科研里就不只是写代码或解释，而是贡献了原创且有用的论证——这是质变信号。

05能力越界，安全就被当成系统工程来做

网络安全 / 生物化学 · Preparedness 框架

生物化学与网络安全能力均被定为"High"

GPT‑5.5 未触及网络安全的"Critical"级，但相比 GPT‑5.4 是明确一步提升。OpenAI 部署了更严格的网络风险分类器（用户初期可能觉得"烦"），并通过 chatgpt.com/cyber 的可信访问机制，让经核验的防御者在更少误拒下使用其能力；关键基础设施单位可申请 GPT‑5.4‑Cyber 等"宽松版"模型。发布前经过近 200 家可信早期合作伙伴的真实用例反馈与内外部红队测试。

OpenAI 的赌注是"民主化访问 + 随能力扩展的护栏 + 检测响应能力"，把网络安全当成需要整个生态共建韧性的"团队运动"，而非单方面封锁。

最值得带走的洞见

  1. 叙事拐点：前沿竞争的焦点从"基准刷分"转向"在长程真实任务中可被托付"——自主性、持久性、少纠错成了核心指标。
  2. 效率即能力：同代延迟下用更少 token 做更多事，让"更强"不再等于"更慢更贵"，显著降低规模化采用门槛。
  3. 能力外溢：编码能力泛化为通用计算机操作，把 AI 的受影响人群从工程师扩展到几乎所有桌面工作者（85% 内部周活、税表/报告自动化为证）。
  4. 科研质变：GeneBench 跃升、Lean 验证的 Ramsey 数新证明，显示模型开始作为"协作科学家"贡献原创论证，而不只是辅助。
  5. 安全前置：生物化学与网络能力被定为 High，OpenAI 选择"可信访问 + 随能力扩展的护栏"而非一刀切封锁——把防御能力的民主化当作长期赌注。
