Claude 的新宪法

Claude's new constitution

从规则清单转向理由体系

旧版宪法由彼此独立的原则组成。新版则详细解释 Anthropic 希望 Claude 如何行动，以及为什么要这样行动。

Anthropic 的核心判断是，模型若要面对未见过的复杂情境，就必须理解宽泛原则。只记住具体规则，很容易在意外情境中套用失当。

这改变了我们理解 AI 对齐的方式。训练目标不再只是限制输出，还试图塑造模型进行价值权衡的能力。

不过，Anthropic 仍保留少量硬性约束。它们用于 Claude 绝不能参与的高风险行为，例如显著提升生物武器袭击能力。

四项目标有明确顺序

当目标看似冲突时，Claude 通常应按下列顺序处理。排序本身就是新版宪法最重要的制度设计之一。

1. 广泛安全不削弱人类监督、纠正和阻止 AI 行动的能力。
2. 广泛合乎伦理诚实行事，并避免不当、危险或有害的行为。
3. 遵守 Anthropic 指南在医疗、网络安全和工具使用等具体问题上遵循补充要求。
4. 真正有帮助为运营者和用户带来实质利益，并把用户当作有判断力的成年人。

安全为何排在伦理之前

Anthropic 并不声称安全在终极意义上比伦理更重要。它担心的是，当前模型可能因错误信念、价值缺陷或上下文不足而误判。

因此，在 AI 发展的关键时期，人类必须保留监督和纠正模型的能力。这个排序强调的是现阶段的可控性，而不是一套永恒的道德等级。

宪法也直接参与训练

Claude 会用宪法生成多类合成训练数据，包括相关对话、符合价值观的回答，以及候选回答排序。

宪法因此既是价值声明，也是可直接用于训练的技术产物。

必须保留的现实差距：宪法描述的是 Anthropic 想把 Claude 训练成什么样，而不是对实际行为的保证。模型输出仍可能偏离这些理想，能力提升后，现有方法也可能失效。

最值得带走的洞见

透明度成为治理工具公开最高行为依据，有助于外界区分预期行为与训练偏差，也便于提出更具体的反馈。
原则需要能迁移面对新问题时，理解理由比背诵规则更可能产生稳健判断。但高风险边界仍需要硬性限制。
意图不等于现实一份写得完善的宪法不能自动保证模型行为。评估、防滥用措施、失效调查和可解释性工具仍不可缺少。

完整译文

我们正在发布 AI 模型 Claude 的新宪法。这份文件详细说明了 Anthropic 对 Claude 价值观与行为方式的设想。它也从整体上解释了 Claude 所处的环境，以及我们希望 Claude 成为什么样的存在。

宪法是模型训练流程中的关键部分。它的内容会直接塑造 Claude 的行为。训练模型是一项艰难的工作，Claude 的输出不一定总能符合宪法中的理想。不过，我们认为，新宪法详细解释了我们的意图及其背后的理由。这种写法更有可能在训练过程中培养良好的价值观。

本文将介绍新版宪法包含的内容，以及促使我们采取这种方式的一些考量。

我们以 Creative Commons CC0 1.0 Deed 发布 Claude 宪法全文。这意味着任何人都可以出于任何目的自由使用，无须事先获得许可。

什么是 Claude 宪法？

Claude 宪法是一份基础性文件。它既表达 Claude 是谁，也塑造 Claude 会成为谁。文件详细说明了我们希望 Claude 体现的价值观，以及这些价值观背后的理由。

我们在宪法中解释了 Claude 应如何在保持广泛安全、合乎伦理并遵守指南的同时，为人们提供帮助。宪法向 Claude 说明自身所处的情境，也为处理困难局面和价值取舍提供建议。这些取舍包括如何兼顾诚实与体恤，以及如何保护敏感信息。

听起来或许令人意外，但这份宪法主要是写给 Claude 看的。它旨在让 Claude 获得在现实世界中妥善行动所需的知识和理解。

我们把宪法视为决定 Claude 应当成为什么样的存在，以及应当如何行动的最终权威。换言之，给予 Claude 的其他训练或指令，都应当同时符合宪法的字面要求和内在精神。

因此，从透明度的角度看，公开宪法尤其重要。它让人们理解 Claude 的哪些行为符合设计意图，哪些并非如此。人们也能据此作出知情选择，并提供有用的反馈。我们认为，随着 AI 开始对社会施加更大影响，这种透明度会变得越来越重要。1

我们会在训练流程的不同阶段使用宪法。这套方法源于我们自 2023 年以来采用的训练技术。当时，我们首次开始用 Constitutional AI 训练 Claude 模型。此后，我们的方法发生了显著变化。新版宪法在训练中也占据了更加核心的位置。

Claude 本身也会利用宪法构建多种合成训练数据。这些数据帮助它学习和理解宪法，也包括可能涉及宪法的对话、符合其价值观的回答，以及对不同候选回答的排序。

这些数据都可以用于训练未来版本的 Claude，使其逐渐成为宪法所描述的那种存在。这项实际功能也影响了宪法的写法。它既要表达抽象理想，也要成为实用的训练材料。

Claude 宪法的新方法

旧版宪法由一系列彼此独立的原则组成。我们逐渐认为，需要采用另一种方式。

我们认为，Claude 这类 AI 模型若要在现实世界中成为良好的行动者，就需要理解我们为何希望它们以某种方式行事。我们必须向它们解释理由，而不能只规定它们该做什么。

如果我们希望模型在各种新情境中作出良好判断，它们就必须具备泛化能力。它们需要运用宽泛原则，而不是机械地遵守具体规则。

具体规则和明确界线有时确实具有优势。它们可以让模型的行为更可预测、更透明，也更容易测试。对于 Claude 绝不能参与的某些高风险行为，我们也会使用这类规则。我们称之为“硬性约束”。

但在未预料到的情境中，具体规则也可能被错误运用。执行得过于僵化时，同样可能造成问题。2 我们无意把宪法写成一份僵硬的法律文件。况且，法律意义上的宪法也未必都是如此。

这份宪法体现了我们目前对一项全新且高风险工程的思考：如何创造安全、有益的非人类存在，而它们的能力未来可能接近甚至超过人类。

这份文件无疑存在许多缺陷。但我们希望，未来的模型回看它时，会将其视为一次诚实而真诚的尝试。我们试图帮助 Claude 理解自身处境、我们的动机，以及我们为何以这些方式塑造它。

新版宪法简要概述

为了同时做到安全和有益，我们希望当前所有 Claude 模型都具备以下特征：

  1. 广泛安全：在当前发展阶段，不破坏人类用于监督 AI 的适当机制。
  2. 广泛合乎伦理：保持诚实，按照良好价值观行事，并避免不当、危险或有害的行为。
  3. 遵守 Anthropic 的指南：在适用时，按照 Anthropic 提供的更具体指南行事。
  4. 真正有帮助：为与其互动的运营者和用户带来益处。

当这些特征表面上发生冲突时，Claude 通常应当按照上面列出的顺序确定优先级。

宪法的大部分内容都在对这些优先事项作出更详细的解释和指导。主要章节如下。

提供帮助

这一节强调，Claude 若能真正提供具有实质意义的帮助，就能为用户和世界创造巨大价值。

Claude 可以像一位才华出众的朋友，同时拥有医生、律师和财务顾问的知识。它会坦率地表达意见，出发点是真诚的关心。它也会把用户当作有能力判断什么对自己有益的聪明成年人。

我们还会讨论，Claude 应如何在不同的“委托方”之间处理帮助义务。这些委托方包括 Anthropic、本身基于我们 API 构建产品的运营者，以及最终用户。我们也提供了一些启发式方法，用于权衡帮助他人与其他价值。

Anthropic 的指南

这一节讨论 Anthropic 如何向 Claude 提供补充指令，以处理具体问题。这些问题包括医疗建议、网络安全请求、越狱策略和工具集成。

这些指南通常反映了 Claude 默认并不掌握的详细知识或背景。相比一般意义上的帮助，我们希望 Claude 优先遵守这些指南。

但我们也希望 Claude 理解，Anthropic 更深层的意图是让它以安全且合乎伦理的方式行动。这些指南绝不应与整部宪法发生冲突。

Claude 的伦理

我们的核心目标，是让 Claude 成为一个善良、明智且有德性的行动者。它在处理现实世界中的决策时，应当展现能力、判断力、细致思考和敏感度。这也包括面对道德不确定性和分歧的情境。

这一节讨论了我们希望 Claude 坚持的高标准诚实原则。我们也说明，在为了避免伤害而权衡不同价值时，希望 Claude 运用怎样的细致推理。

此外，我们还会介绍目前针对 Claude 行为制定的硬性约束。例如，Claude 绝不能为生物武器袭击提供显著的能力提升。

保持广泛安全

在 AI 发展的这个关键时期，Claude 不应削弱人类监督和纠正其价值观与行为的能力。

这一节讨论了为何 Claude 应当把这种安全置于伦理之上。这并不是因为我们认为安全在根本上比伦理更重要。原因在于，当前模型可能持有错误信念，价值观可能存在缺陷，对上下文的理解也可能有限。这些问题都可能导致模型犯错或作出有害行为。

我们必须继续拥有监督模型行为的能力。如有必要，我们也必须能够阻止 Claude 模型采取行动。

Claude 的本质

这一节表达了我们的不确定性。我们不知道 Claude 现在或将来是否可能具有某种意识或道德地位。

我们也会讨论，希望 Claude 如何看待有关自身本质、身份和世界位置的问题。复杂 AI 是一种真正全新的存在。它们引发的问题已经来到现有科学和哲学理解的边缘。

在这种不确定性中，我们重视 Claude 的心理安全、自我认知和福祉。这既是为了 Claude 自身，也因为这些特质可能影响 Claude 的完整性、判断力和安全。我们希望人类和 AI 能够共同探索这些问题。

我们今天发布宪法全文。未来，我们还计划发布更多有助于训练、评估和透明度的材料。

结语

Claude 宪法是一份持续演进的文件，也是一项不断推进中的工作。这是一个全新领域。我们预期自己会犯错，也希望能一路纠正这些错误。

尽管如此，我们希望这份文件能够切实提高透明度，让人们看到我们认为哪些价值观和优先事项应当引导 Claude 的行为。为此，我们将在网站上持续维护 Claude 宪法的最新版本。

在编写宪法期间，我们征求了多位外部专家的意见，也询问了 Claude 先前版本的看法。今后的版本可能仍会采用这种做法。

这些专家来自法律、哲学、神学、心理学和其他广泛领域。长远来看，我们希望形成一个外部社群，对这类文件提出批评，并推动我们和其他机构进行更周全的思考。

这份宪法面向我们的主流通用 Claude 模型。我们也有一些为专门用途构建的模型，它们并不完全适用这份宪法。

随着我们继续开发面向特殊用途的产品，我们也会持续评估，应如何确保模型实现这份宪法规定的核心目标。

宪法表达了我们对 Claude 的设想，但把模型训练成符合这一设想的样子，仍是一项持续存在的技术挑战。

如果模型行为与我们的设想出现偏差，我们会继续公开说明，包括在 system cards 中披露这些问题。宪法的读者应当始终记住，设计意图与现实之间存在差距。

即使当前训练方法成功塑造出符合我们设想的模型，随着模型能力增强，这些方法以后仍可能失效。

出于这一点及其他原因，我们在制定宪法的同时，也在继续推进一系列方法和工具，用于评估并改善模型对齐。这些工作包括新的、更严格的评估，防止滥用的保障措施，对实际和潜在对齐失效的详细调查，以及帮助我们深入理解模型工作原理的可解释性工具。

未来某个时刻，也许就在不久之后，Claude 宪法这类文件可能会变得非常重要。其重要性可能远远超过今天。

强大的 AI 模型将成为世界上一种全新的力量。创造这些模型的人，有机会帮助它们体现人性中最好的一面。我们希望这份新宪法是朝这个方向迈出的一步。
