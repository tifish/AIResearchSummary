用 AI 越多的人，反而越不怕 AI

来源 Anthropic ｜ 经济指数报告 ｜ 2026-06-26

一句话总括：这份报告的真正贡献不是又一张使用排行榜，而是把"小时级使用日志"与"近万人主观调查"打通，得出一个反常识结论——越是把整段任务自动化交给 Claude 的人，对自己的薪资、饭碗和职业意义反而最乐观。它把"重度用 AI 的人最该焦虑"这一直觉翻了过来。

方法升级从七天快照到小时级脉搏

过去每期经济指数只取一次 7 天样本；这一版改为每天持续采样，能看到按小时、按工作日的节律，并新增了"产出物（artifact）"分类器，还首次接入了 2026 年 4 月启动的用户调查。三项改动共同把"AI 如何渗入经济生活"看得更清。一个直观例证：报税相关对话在美国截止日前一天（4 月 14 日）是 5 月平均日的 8 倍，4 月 16 日骤降。

结论：当 Claude 从"聊天框"变成能连跑数小时的 agent，旧的测量方式已经看不全它的经济足迹。

使用即生活工作日是公事，周末是私事

使用节律几乎是真实生活的镜像。个人用途占比从工作日约 35% 跳到周末近 50%，话题从商务邮件、营销文案、幻灯片，转向情感支持、医疗问题和投资建议——高收入国家这种切换最明显。一天之内同样有节奏：早 7 点问新闻，傍晚问食谱，黎明前问助眠。

更有意思的是夜晚和周末仍在工作的人：这些时段的任务更偏高薪职业（如营销经理、程序员），而电话销售、文书等低薪职业的占比下降——剔除计算机类职业后，这一倾斜依然成立。

结论：使用日志里刻着真实世界的作息，AI 的"工作—生活"边界与人类高度同步。

产出画像九成对话都留下了东西

分类器识别出 93% 的对话产出了某种 artifact。最常见的是解释（17%）、文档与报告（15%）、指导（11%）。对话型产出（解释/指导）和书面交付物（文档/演示）各约占三分之一，代码与技术工作约占六分之一。同一种产出可能是工作也可能是私事：超 80% 的创意写作、指导、食谱是个人用途，而营销内容（80%）、博客文章（81%）、数据库查询（82%）几乎都是工作。

结论："产出是什么"和"产出为了谁"是两回事——artifact 让 Claude 的工作第一次变得可清点。

算力 = 价值越值钱的活，越烧 token

每段对话消耗的 token，随其对应职业的工资水平上升。营销经理时薪约为编辑的两倍（$80 vs $37），对应对话消耗的 token 约为 2.5 倍；建 app 的对话用掉中位数 3 倍以上的 token，而一次解释只用约五分之一。约 44% 的"工资—算力"梯度，由产出类型的差异解释。

关键在于：Claude 产出更多，并不意味着人类参与更少——两者同步上升。如果人类在最高价值的任务上始终在场，这个图景更像"增强劳动"，而非"替代劳动"。

结论：更值钱的产出确实更贵，但贵在"人机一起投入更多"，而不是人退场。

产品 > 模型同一件事，Claude Code 放手更多

用 1–5 分衡量 Claude 的自主决策程度：在展示的 31 类产出中，有 26 类 Claude Code 的自主性高于 chat/Cowork，平均高 0.37 分。最生动的例子是博客文章——chat/Cowork 中位需要 13 轮往返，而 Claude Code 中位只有 1 次人类提示。约三分之二的差异来自"同样的任务被更多地委托出去"。

这并非单纯因为模型更强：Claude Code 用 Opus 的比例（54%）远高于 chat/Cowork（10%），但即便只比同样用 Sonnet 的对话，Claude Code 仍高 0.26 分。是产品形态、而非底层模型，在更大程度上决定了人类愿意放手多少。而自主性最高的产出，恰恰也是最烧算力的（r = 0.68）。

结论：曾经"需要持续判断、难以自动化"的创造类工作，正随着 agent 型产品开始被委托出去。

主观调查自动化用得越多，越乐观

把约 9,700 名受访者的回答与其真实使用打通后，最反直觉的发现出现了：在薪资、饭碗、再就业、意义、自主、人际六个维度上，自动化使用占比越高的人，对 AI 明年影响越乐观，其中对"未来薪资"和"找工作能力"的正向预期提升最大。控制了 Claude.ai 使用资历（早/晚采用者的代理变量）后，结论不变。

配套的事实是：大多数人确实在受益——报告生产力提升的比例为速度 86%、范围 82%、质量 69%；68% 觉得自己学到更多，57% 觉得技能更值钱。常见的"把任务交给 AI = 放弃思考、技能退化"的担忧，在数据里没有出现：重度委托者的学习率与其他人持平（但作者也提醒这是自评，不能据此排除技能侵蚀）。

结论："离 AI 越近越焦虑"并不普遍成立；真正放手用 AI 的人，反而更相信自己的前景。

谁更担心谁替别人、替新人担心更多

人们对未来进步的"速度"预期惊人一致（符合"水涨船高"），但对这对自己意味着什么则分歧很大。越早期职业、越年轻的人认为 AI 能做的工作份额越高、对失业越担忧；高收入国家的人则觉得 AI 现在能替自己做的份额低约 10 个百分点。值得注意的是，人们替他人比替自己更担心：超三分之一认为初级同事一年内失业概率高于 60%。性别差异也很明显——女性仅占打通样本的 12%，其 Claude Code 使用占比低 6.3 个百分点、自动化占比低 7.3 个百分点，更倾向迭代式协作。

结论：焦虑集中在"新人"和"别人"身上；而对自身前景，越有经验、越富裕的人越淡定。

他们的愿望不是被替代，是被解放

调查以一道"大胆想象：十年后被 AI 重塑的经济是什么样"收尾。最高频的三个主题是：增强——超半数希望与 AI 协作做有意义的工作、让职业仍然重要；自动化——刚过半数希望 AI 接管枯燥部分，腾出自由时间；共享繁荣——约三分之一希望 AI 的收益被广泛分享。

结论：哪怕是最贴近 AI 前沿的重度用户，对未来的期待也压倒性地落在"协作"而非"替代"上。

最值得带走的洞见

  1. 反直觉主结论：自动化≠焦虑。在全部六个维度上，把整段任务交给 AI 的人对自己前景最乐观，控制"采用早晚"后依然成立。
  2. 图景偏"增强"而非"替代"。高价值对话里 Claude 产出更多（1.34×）的同时，人类参与也更多（1.53× 轮次）——人没退场，所以更像增强劳动。
  3. 决定放手程度的是产品，不是模型。同样用 Sonnet，Claude Code 仍比 chat 多 0.26 分自主性；博客类任务从 13 轮往返压缩到 1 次提示。
  4. 使用日志映射真实生活节律。周末个人用途从 35% 升到 50%，报税对话截止日前飙到 8 倍——小时级数据让经济活动的脉搏可见。
  5. 担忧投向新人，期待投向协作。10% 担心自己失业、却有逾三分之一认为初级同事高危；而所有人的十年愿望，最多落在与 AI 协作、解放枯燥、共享繁荣。

注：调查样本不代表总体人群——计算机/数学类职业约占受访者 30%（美国就业仅 4%），管理类约 23%（就业 7%），且过滤掉了低频用户。文中"自动化占比"指对话呈"指令式/反馈循环式"、由 AI 基本独立完成的比例。所有分析均基于隐私保护分类器（转写仅由另一实例的 Claude 读取）。
