一句话总括

GPT-5.6：随你的雄心扩展的前沿智能

GPT-5.6: Frontier intelligence that scales with your ambition

GPT-5.6 的真正贡献，是把更强模型、更低单位成本、程序化工具调用和多代理并行组合成一种更接近“端到端工作系统”的 AI 能力。

核心变化不是更会聊天，而是更会完成工作

GPT-5.6 分成 Sol、Terra 和 Luna 三个层级。Sol 面向最高能力，Terra 面向日常平衡，Luna 面向最低成本。

这让用户不必把所有任务都交给最贵模型。简单任务可以省钱，困难任务可以切换到 max 或 ultra。

结论 模型竞争正在从“单次回答质量”转向“单位成本完成多少真实工作”。

效率提升改变了部署逻辑

Sol 在 Agents’ Last Exam 得分 53.6，比 Claude Fable 5 高 13.1 分。中等推理下仍领先 11.4 分，成本约为四分之一。

Terra 和 Luna 也被定位为高性价比选择。它们在部分测试中超过 Fable 5，估算成本约为十六分之一。

洞见 如果这些数字能在实际业务中复现，AI 能力会更容易进入高频、低利润的工作流。

编码能力的重点，是更少返工和更少等待

能力面关键数据为什么重要
编码代理 Sol 在 Coding Agent Index 得分 80，比 Fable 5 高 2.8 分。 它说明模型不只是会写代码，也更能处理终端、真实仓库和长链路实现。
成本与速度 输出 token 少于一半，耗时少于一半，成本约低三分之一。 编码代理的瓶颈常在等待和预算。效率提升会直接影响产品可用性。
Programmatic Tool Calling 模型可写轻量程序处理工具结果，减少轮次和无效上下文。 这把模型从“读每一条工具输出”推向“主动筛选中间结果”。

ultra 把复杂任务拆成并行工作流

max 给模型更多推理时间。ultra 默认协调 4 个代理并行工作，并可在 API 中通过 multi-agent beta 构建类似体验。

1. 分解把任务拆成多个可并行的研究、实现或验证分支。
2. 执行多个代理同时推进，减少长任务的墙钟时间。
3. 合成根代理汇总结果，保留关键证据和可交付物。
4. 取舍用更多 token 换更高成功率和更快完成时间。

这改变了读者对 AI 影响的理解：前沿模型正在接近团队式执行，而不只是单个助手的顺序回答。

知识工作的提升集中在可交付物

GPT-5.6 在 BrowseComp 达到 92.2%，在 OSWorld 2.0 达到 62.6%。在 OSWorld 上，它超过 Opus 4.8，同时少用 85% 输出 token。

文章还强调它更会做演示文稿、文档和电子表格。尤其是在参考模板和母版规则存在时，它能更稳定地继承版式、层级和数据呈现方式。

结论很直接：企业用户关心的不是模型是否“懂”，而是它能否交出可编辑、少返工、能分享的成果。

当模型能检查渲染结果并修正细节，设计能力就不再只是生成代码，而是进入真实交付阶段。

网络安全和科学能力同步上升

在 ExploitBench 2 中，GPT-5.6 得分 73.5%，GPT-5.5 为 47.9%。在 ExploitGym 3 中，两小时通过率从 15.1% 提升到 24.9%，六小时达到 33.7%。

SEC-Bench Pro 得分 71.2%，GPT-5.5 为 45.8%。这些数据表明模型更能处理真实漏洞、概念验证和长链路安全任务。

科学部分也出现提升，尤其是在基因组学、定量生物学、生命科学流程和化学任务上。文章称这些提升没有跨过生物或网络安全的 Critical 阈值。

风险不是附录，而是发布条件

OpenAI 使用训练内防护、实时检查、连续监控、账户级执行和推理监控来分层控制风险。

Sol 的网络安全防护比以往模型多拦截约 10 倍潜在有害活动。代价是良性使用可能遇到更多摩擦。

这说明越强的模型越需要按身份、用途和风险分配能力，而不是简单开放或简单封锁。

内部研究加速是最值得关注的信号

内部测试
GPT-5.6 期间，OpenAI 活跃研究员的日均输出 token 超过 GPT-5.5 最高水平的两倍。
过去六个月
内部编码推理占研究算力的份额增长 100 倍，内部代理式 token 使用增长约 22 倍。
RSI 评估
在衡量递归自我改进进展的一组内部评估中，GPT-5.6 Sol 比 GPT-5.5 提高 16.2 分。

这些指标不等于研究突破本身，但说明 AI 已经深度嵌入模型研发流程。读者应关注的变化，是 AI 正在成为改进 AI 的基础设施。

可用性和价格给出了产品化边界

GPT-5.6 从发布日起进入 ChatGPT、Codex 和 OpenAI API，并在 24 小时内逐步扩展可用性。ChatGPT Work 和 Codex 用户可按计划选择 Sol、Terra、Luna 与不同 effort。

API 定价按每百万 token 计算：Sol 为 5 美元输入、30 美元输出；Terra 为 2.50 美元输入、15 美元输出；Luna 为 1 美元输入、6 美元输出。

缓存策略也更可预测，包括显式 cache breakpoint 和 30 分钟最短缓存生命周期。缓存写入按未缓存输入价的 1.25 倍计费，缓存读取继续享受 90% 折扣。
